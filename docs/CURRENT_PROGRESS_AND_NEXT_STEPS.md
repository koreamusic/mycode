# Pop WiFi MV Studio — Current Progress and Next Steps

작성일: 2026-04-25
문서 목적: 현재 개발 상태와 다음 작업을 Codex/다음 개발자가 바로 이어받기 위한 상태 기록

---

## 1. 현재 확정된 개발 기준

- Pop WiFi MV Studio는 PC 중심 제작 스튜디오다.
- 모바일은 제작 중심이 아니라 확인/검수용이다.
- 원페이지 개발은 금지한다.
- 하나의 앱 안에서 페이지/모듈을 분리한다.
- 숏츠 프리셋은 디자인 기준 원형이다.
- 롱폼 16:9 프리셋은 숏츠 프리셋을 기반으로 재설계한다.
- 숏츠 프리셋을 단순 확대/복사해서 롱폼에 적용하면 실패다.
- 프리셋은 쉽게 추가/삭제/비활성화할 수 있어야 한다.
- FAQ 페이지를 만들어 전체 구조와 프리셋 제작법을 문서화한다.

## 10. 최종 목표

최종 목표는 PC에서 다음 흐름이 가능한 수준이다.

```text
프로젝트 생성
→ 음원/이미지/가사 연결
→ 숏츠 또는 롱폼 프리셋 선택
→ 가사 마킹
→ 720p 랜딩 검수
→ 최종 렌더 큐 등록
→ FFmpeg 출력
→ 결과 확인
```

모바일은 이 결과를 확인하고 숏츠 화면을 검수하는 보조 화면으로 유지한다.
