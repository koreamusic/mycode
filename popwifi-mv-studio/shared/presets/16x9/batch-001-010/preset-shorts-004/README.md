- 9x16 중앙 트위스트 박스 → 16x9 **수평 4컬럼 그리드** 구도로 재설계
- 컬러 레이어를 수직 열(column)로 분배, 각각 독립적 위상으로 진동
- 타이틀: 좌측 앵커 배치 (동적 배경 위 가독성 확보)
- CTA: 3개 컬러 칩을 수평 배열
- 수평 크로스 라인으로 구도 분할
