원본: `9x16/batch-001-010/preset-shorts-002`

## 재설계 원칙

- 9x16 수직 슬라이딩 테두리 → 16x9 **수평 슬라이딩 라인** 구도로 재설계
- 상단/하단 수평 라인이 반대 방향에서 슬라이드인
- 좌측 수직 단선이 앵커 역할
- 타이틀: 하단 좌측 third 배치 (16:9 영상에서 더 자연스러운 위치)
- 시네마틱 letterbox 느낌 강조

## 재설계 금지 항목

- 9x16 수직 라인을 그대로 가져오지 않는다
- 타이틀을 정중앙에 배치하지 않는다 (9x16 방식)
