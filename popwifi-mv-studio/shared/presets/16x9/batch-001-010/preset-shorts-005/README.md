원본: `9x16/batch-001-010/preset-shorts-005`

## 재설계 원칙

- 9x16 수직 레이어 → 16x9 **좌측 앵커 수평 라인** 구도로 재설계
- 배경: 밝은 페이퍼 크림 (9x16 어두운 버전과 반전 — 와이드에서 더 고급스러움)
- 상단/하단 수평선이 반대 방향에서 슬라이드인
- 좌측 수직 앵커선이 타이틀 기준점
- 타이틀: 좌측 정렬 (프리미엄 에디토리얼 스타일)
- 하단바: 좌측 정렬 텍스트 (9x16 중앙 정렬과 차이)

## 재설계 금지 항목

- 9x16 어두운 배경을 그대로 가져오지 않는다
- 타이틀을 중앙 좌측이 아닌 정중앙에 배치하지 않는다
